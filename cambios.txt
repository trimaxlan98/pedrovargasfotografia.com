Actúa como un **senior frontend engineer + UI/UX designer especializado en landing pages premium para eventos**.

Estoy desarrollando un **sistema de invitaciones digitales tipo landing page vertical optimizada para celular**.

Necesito que revises y mejores el diseño actual aplicando **mejores prácticas de frontend moderno** y usando tus **plugins o herramientas de análisis de UI/UX, layout, responsive design y accesibilidad**.

Objetivo del proyecto:

Crear invitaciones digitales tipo **experiencia de scroll (storytelling)** similares a una **landing page elegante o revista digital**, optimizada principalmente para **dispositivos móviles**.

Quiero que analices el código del proyecto y realices **refactor y mejoras visuales** para cumplir con los siguientes requisitos:

---

# 1. Estructura general

La invitación debe sentirse como una **experiencia al hacer scroll**, no como una tarjeta simple.

Debe funcionar como:

landing page vertical
scroll continuo
secciones claramente separadas
mucho espacio visual
estilo elegante y moderno

Usa buenas prácticas como:

section blocks
max-width controlado
spacing generoso
layout limpio

---

# 2. Portada / Hero

La primera sección debe ser un **Hero visual impactante** que incluya:

Nombre del festejado (elemento principal y más grande)
Texto: "Mis XV Años"
Fecha del evento
Countdown dinámico

El nombre debe dominar visualmente la pantalla.

Opcionalmente puedes agregar:

animaciones suaves
background elegante
partículas sutiles
gradient overlays

---

# 3. Jerarquía tipográfica

Mejora la jerarquía visual usando escalas claras:

H1 → Nombre
H2 → Títulos de sección
H3 → subtítulos
P → texto descriptivo

Usa:

tipografías elegantes
espaciado entre líneas adecuado
tracking y weights correctos

---

# 4. Secciones necesarias

La landing debe dividirse en bloques claros:

1. Portada
2. Foto principal
3. Mensaje de invitación
4. Dónde y cuándo
5. Ceremonia
6. Recepción
7. Código de vestimenta
8. Galería
9. Padres y padrinos
10. Regalos
11. Instagram
12. Confirmación de asistencia
13. Mensaje de invitados
14. Cierre

Cada sección debe estar claramente separada visualmente.

---

# 5. Ceremonia y Recepción

Cada evento debe mostrarse como una **card visual elegante**.

Cada card debe incluir:

foto del lugar
fecha
hora
dirección
botón VER MAPA

El botón debe abrir Google Maps.

---

# 6. Botones de interacción

Agregar botones visualmente claros para:

Ver mapa
Confirmar asistencia (WhatsApp)
Ver galería
Compartir invitación

Usa estilos modernos como:

rounded buttons
hover effects
microinteractions

---

# 7. Galería

Crear una galería elegante con:

grid responsive
bordes redondeados
espacio entre fotos
lightbox al abrir imagen

---

# 8. Diseño visual

El diseño debe sentirse:

elegante
femenino
moderno
estilo revista

Aplicar:

más whitespace
separadores sutiles
layout limpio
micro animaciones

---

# 9. Espaciado

Aumentar significativamente:

padding entre secciones
márgenes laterales
separación entre componentes

Evitar layouts comprimidos.

---

# 10. Arquitectura del sistema

Cada sección debe ser un **bloque editable en el sistema**.

El sistema debe permitir:

activar / desactivar secciones
subir fotos
editar textos
cambiar colores
cambiar tipografías
agregar mapas
agregar música

---

# 11. Animaciones

Implementar animaciones suaves:

fade-in al hacer scroll
aparición progresiva de imágenes
contador animado

Evitar animaciones pesadas.

---

# 12. Responsive design

Optimizar principalmente para:

mobile first

pero también compatible con:

tablet
desktop

---

# 13. Código

Si encuentras problemas en el código:

corrige estructura
refactoriza componentes
mejora CSS
optimiza layout

Explica brevemente los cambios realizados.
